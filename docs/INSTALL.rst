ploneun.regions Installation
----------------------------

To install ploneun.regions using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ploneun.regions`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ploneun.regions

* Re-run buildout, e.g. with:

    $ ./bin/buildout
